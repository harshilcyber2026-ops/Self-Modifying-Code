Background

Self-modifying code (SMC) decrypts or modifies parts of itself at runtime. Malware authors use it to evade static AV scanners. In this CTF, the binary stores the flag encrypted — static tools show only noise and decoys.


Goal: Find the encrypted buffer and the XOR key used to decrypt it. The key is visible in the disassembly of the decryption loop.
